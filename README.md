## Features

- Maven multi-module project
- Libraries
 - utils:           guava, commons-lang3
 - cli:             jcommander
 - template engine: jtwig
- Tech stack: java 8 / maven
- Tools: git flow

*solution-framework* is *TDD* compatibile - it creates *red* tests, you'll have to go *green* yourself.
